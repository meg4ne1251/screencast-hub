## 概要
プロジェクターのNetflix DRM（低画質）問題を根本から解決し、最高画質かつ快適な専用UIを備えたメディアセンターを自作する。
DRM制限をクリアできるWindows環境を裏で動かしつつ、表向きは完全に「自作のスマートテレビ」として振る舞わせる。さらに、AirPlayやChromecastの受信機能を持たせ、手元のデバイスからのシームレスな画面キャストを実現する。

## システム構成
* **ハイパーバイザ:** Proxmox VE
* **ゲストOS:** Windows 11 VM（例: `ScreencastHub` や `任意のVM名` などの命名規則に合わせた専用機）
* **映像出力:** GPUパススルーを用いて、VMからプロジェクターへ直接HDMI出力
* **UI (フロントエンド):** AstroやReactで構築した自作ポータルサイト（SPA）
* **キャスト受信:** AirServer (Windows Desktop Edition) または Reflector 4
* **操作系:** Unified Remote ＋ 物理リモコン（Bluetoothメディアリモコン または FLIRC）
* **操作/キャスト元:** スマートフォン、MacBook

## 実装フェーズとタスク一覧

### Phase 1: 基盤構築（Proxmox & Windows VM）
- [ ] Proxmox上でのWindows 11 VM作成
- [ ] ネットワーク設定（VMのIPアドレス固定化）
- [ ] **GPUパススルーの設定**
  - [ ] Proxmox側のIOMMU有効化・分離設定
  - [ ] VMへのPCIデバイス追加とドライバインストール
  - [ ] プロジェクターへのHDMI出力テスト（画面が正常に映るか確認）
- [ ] Windows 11の自動サインイン設定（パスワード入力なしで起動させる）
- [ ] NetBoxへのIPアドレス・PCIeパススルー等のインフラ構成情報のドキュメント化

### Phase 2: キャスト環境の構築（AirPlay / Chromecast）
- [ ] Windows VMに `AirServer` (または `Reflector 4`) をインストール
- [ ] 受信機としての名前設定（例: `ScreencastHub-Display` など）
- [ ] Windowsのスタートアップへの登録（バックグラウンド常駐）
- [ ] **ネットワーク越しのキャスト確認**
  - [ ] 同じセグメント内でのキャストテスト
  - [ ] （※サーバー用VLAN等でセグメントを分けている場合）L3スイッチやルーター側でのmDNSリフレクター（Avahi等）の設定・中継テスト
  - [ ] MacBookやスマートフォンからキャストし、全画面でポップアップ表示されるか確認

### Phase 3: 操作環境の構築（Unified Remote）
- [ ] Windows VMに `Unified Remote Server` をインストール
- [ ] ファイアウォールの設定（TCP/UDP 9512番ポートの開放）
- [ ] セキュリティ設定（必要に応じてパスワード設定）
- [ ] スマホ / MacBookにクライアントアプリをインストールし、接続テスト
- [ ] （オプション）物理リモコンの接続とキーバインドの確認

### Phase 4: 専用UIの開発（自作ポータルサイト）
- [ ] Astro / Reactを用いたフロントエンドの開発
  - [ ] 十字キー（矢印キー）とEnterで操作できるフォーカス制御の実装
  - [ ] 各種ストリーミングサービスへの大きなリンクボタン（UI）の作成
- [ ] アプリ起動用URIスキームの実装
  - 例: Netflixネイティブアプリを叩く `netflix://` リンクの設置
- [ ] ローカル環境でのホスティング設定

### Phase 5: Kiosk化・結合テスト（最終仕上げ）
- [ ] Windowsのスタートアップ設定
  - [ ] ブラウザ（Edge/Chrome）のショートカットを作成
  - [ ] 起動オプションに `--kiosk "http://localhost:xxxx"`（自作ポータルのURL）を付与
- [ ] **シームレス挙動の総合テスト**
  - [ ] VM起動 → 自動サインイン → 自作ポータルが全画面で立ち上がる
  - [ ] リモコンでNetflixを起動し、再生後にポータルへ戻れるか確認
  - [ ] ポータル待機中にAirPlayを飛ばすとキャスト画面が最前面に割り込み、キャスト終了時にポータルへ自動復帰するか確認

## 今後の拡張アイデア (Memo)
* **カスタムリモコンの作成:** Unified Remoteの `Custom Remotes` 機能（XML/Lua）を使って、スマホ側に「自作ポータル専用の操作パネル（Netflix一発起動ボタンなど）」を作る。
* **音声入力の活用:** Unified Remoteの音声入力機能を利用した、検索窓への文字入力のシームレス化。
* **自動電源連動:** Wake-on-LAN (WoL) やスマートプラグを絡めて、プロジェクターの電源ONに合わせてVMを起動させる仕組みの構築。
