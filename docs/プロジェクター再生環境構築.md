# プロジェクター購入時のチェックリスト

## 1. 中古プロジェクター特有の注意点

* **ランプの稼働時間と寿命**
  * 水銀ランプは寿命が短い（数千時間程度）ため、必ず「使用時間」を確認する。
  * 予算が許せば、寿命が長い（2〜3万時間）LED/レーザー方式を選ぶと安心。
* **「リアル（ネイティブ）解像度」の確認**
  * 「対応解像度1080p」という表記に騙されない。
  * 実際のパネル解像度である「リアル解像度」または「ネイティブ解像度」が1080p（フルHD）以上のものを選ぶ。
* **明るさ（ANSIルーメン）の基準**
  * 独自の誇張されたルーメン数ではなく、世界基準の「ANSIルーメン」表記で比較する。
  * 目安：暗い部屋なら200〜500 ANSIルーメン、少し明るい部屋なら1000 ANSIルーメン以上。
* **接続端子とインターフェース**
  * HDMI端子があるか確認する。
  * NetflixやPrime Videoなどを見る場合、著作権保護規格の「HDCP」に対応しているか確認する。
* **動作保証と返品ポリシー**
  * 個人間取引（フリマアプリ等）は初期不良のリスクがあるため慎重に。
  * 返品保証のある中古ショップ（ハードオフなど）や、保証付きの専門店・整備済み品を優先する。

## 2. プロジェクター全般（新品・中古共通）の注意点

* **投影距離と部屋の広さ**
  * 100インチの大画面にするには、壁から約2.5〜3メートルの距離が必要。
  * 部屋が狭く距離が取れない場合は「短焦点モデル」を選ぶ。
* **設置場所と「台形補正機能」**
  * 真正面に置けない（斜めから投影する）場合は、「上下左右（タテヨコ）の台形補正機能」が必須。※タテのみの補正しかできない安価なモデルに注意。
* **ファンの騒音（静音性）**
  * 排気音が視聴の邪魔にならないよう、騒音レベルが「30dB以下」のモデルを目安にする。
* **音響環境（内蔵スピーカー）**
  * 本体内蔵スピーカーの有無を確認する。
  * 音質にこだわる場合は、外部スピーカーやイヤホン（Bluetoothまたは有線）を繋ぐ前提で構成を考える。
* **スマート機能（OS）の有無**
  * プロジェクター単体でYouTubeやNetflixを見たい場合は「OS搭載（Android TVなど）」を選ぶ。
  * OS非搭載の場合は、PCを繋ぐか「Fire TV Stick」などを別途用意する必要がある。


# 自作最強スマートプロジェクター環境 構築プロジェクト

## 概要
プロジェクターのNetflix DRM（低画質）問題を根本から解決し、最高画質かつ快適な専用UIを備えたメディアセンターを自作する。
DRM制限をクリアできるWindows環境を裏で動かしつつ、表向きは完全に「自作のスマートテレビ」として振る舞わせる。さらに、AirPlayやChromecastの受信機能を持たせ、手元のデバイスからのシームレスな画面キャストを実現する。

## システム構成
* **ハイパーバイザ:** Proxmox VE
* **ゲストOS:** Windows 11 VM（例: `ScreencastHub` や `任意のVM名` などの命名規則に合わせた専用機）
* **映像出力:** GPUパススルーを用いて、VMからプロジェクターへ直接HDMI出力
* **UI (フロントエンド):** AstroやReactで構築した自作ポータルサイト（SPA）
* **キャスト受信:** AirServer (Windows Desktop Edition) または Reflector 4
* **操作系:** Unified Remote ＋ 物理リモコン（Bluetoothメディアリモコン または FLIRC）
* **操作/キャスト元:** スマートフォン、M1 Pro MacBook
